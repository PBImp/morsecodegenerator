Colorful Morse Code Generator - Directions

1. Open the file `morse_generator.html` in your web browser (Chrome, Firefox, Edge, etc.).

2. Type your message into the text box at the top.

3. Press "Enter" to automatically generate the Morse code in the output box.

4. Each symbol is colored:
   - Green dot (.) 
   - Red dash (-)
   - Gray space (/)

5. To play your message as Morse:
   - Adjust the speed (milliseconds per dot) if desired.
   - Click the "Play Morse" button.
   - The light circle will flash for each symbol, and you will hear beep sounds.

6. You can clear the text box by deleting the text manually.

Tips:
- Letters, numbers, and many special characters are supported.
- Spaces between letters are automatically included.
- Experiment with speed to get the best timing for beeps and light flashes.

These directions were made by ChatGPT and PBImp who also helped make the generator.
